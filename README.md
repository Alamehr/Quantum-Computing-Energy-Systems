# Quantum Computing in Energy Systems

This repository contains my coursework, programming assignments, and implementations for the graduate-level course **Quantum Computing in Energy Systems** at Sharif University of Technology.

## Key Frameworks Used
- **PennyLane** & **Qiskit** (Quantum circuit simulation and optimization)
- **Python** (NumPy, SciPy)

## Contents
 **Assignments:** Variational Quantum Eigensolvers (VQE) and hybrid quantum-classical algorithms applied to energy systems.
